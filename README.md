# WebDevHub

WebDevHub is a hub for web developers, giving them access 
to a plentitude of tools to help them with their developing 
needs.

### Planned Features
- Code Beautifier (HTML, CSS, JS, PHP etc)
- XML to JSON converter
- Hex to css `filter:` converter
